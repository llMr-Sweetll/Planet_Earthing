# Code Of Conduct

Planet_Earthing is a public research project. The work benefits from disagreement, but only when disagreement is precise, respectful, and grounded in evidence.

## Expected Behavior

- Be kind and direct.
- Critique claims, methods, and assumptions rather than people.
- Welcome careful skepticism.
- Respect privacy and consent boundaries.
- Credit sources and contributors.
- Make space for newcomers to ask basic questions.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks.
- Publishing private participant data.
- Misrepresenting exploratory work as proof.
- Knowingly adding fabricated or misleading sources.
- Dismissing safety, privacy, or ethics concerns as bureaucracy.

## Reporting

Open a private report with the repository owner if possible. If the issue involves public documentation, open a GitHub issue with only the non-sensitive details.

## Enforcement

Maintainers may edit or remove content, close discussions, block contributors, or restrict access when behavior harms the project or participants.
