# Synthetic Pet The Plant Dataset

This directory contains synthetic data for testing code paths only.

It is not empirical plant data.
It must not be used to support biological claims.

Generated with:

```bash
PYTHONPATH=src python3 scripts/generate_synthetic_dataset.py
PYTHONPATH=src python3 scripts/build_feature_table.py
```

Contents:

- `sessions.csv`: synthetic session metadata summary.
- `SYN001/` to `SYN004/`: synthetic session directories.
- `feature_table.csv`: baseline-corrected demo feature table.

The synthetic signal intentionally includes small condition-shaped pulses so feature extraction can be tested. Those pulses are generated by code, not observed from a plant.
