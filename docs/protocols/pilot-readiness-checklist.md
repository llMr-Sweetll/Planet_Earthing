# Pilot Readiness Checklist

Version: 0.1.0

The pilot may start only when every item below is complete.

## Biological Setup

- [ ] Species and cultivar are selected.
- [ ] Plant source lot is documented.
- [ ] Plant inclusion/exclusion criteria are applied.
- [ ] Plants are acclimated for at least 14 days.
- [ ] Watering and light schedules are stable.

## Instrumentation

- [ ] Electrical channel passes dummy-load and plant-baseline checks.
- [ ] Pressure/touch channel has force calibration.
- [ ] Microclimate sensors are cross-checked.
- [ ] Event markers align with all streams.
- [ ] Camera framing verifies hand path and accidental contact.
- [ ] Phase 0 calibration report is complete.

## Protocol

- [ ] Consent/privacy packet is approved for use.
- [ ] Randomization schedule is generated.
- [ ] Blind-label workflow is tested.
- [ ] Session metadata validates.
- [ ] Failure/exclusion rules are understood by operators.
- [ ] Raw data storage location is access-controlled.

## Analysis

- [ ] Synthetic data pipeline runs end to end.
- [ ] Feature table generation works on demo data.
- [ ] Grouped holdout strategy is implemented or documented.
- [ ] Confound-only model plan is documented.
- [ ] Preregistration freeze candidate is reviewed.

## Decision

Choose one:

- [ ] Pilot is ready to start.
- [ ] Pilot is blocked pending listed actions.
