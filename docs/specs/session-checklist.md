# Session Checklist

Version: 0.1.0

## Before Session

- [ ] Confirm plant ID and session ID.
- [ ] Confirm randomized condition and blind label.
- [ ] Confirm no unlogged room entry in the pre-session window.
- [ ] Confirm soil moisture and care events.
- [ ] Check electrode placement and cable strain relief.
- [ ] Record electrode impedance.
- [ ] Zero and verify pressure glove.
- [ ] Confirm microclimate sensors are logging.
- [ ] Confirm audio/video recording and timestamp sync.
- [ ] Confirm participant ID and consent status.
- [ ] Confirm participant followed scent/glove/script restrictions.

## During Baseline

- [ ] No person in room unless condition requires presence.
- [ ] No touching, watering, pruning, fan change, or sensor adjustment.
- [ ] Event marker ready.

## During Interaction

- [ ] Start event marker.
- [ ] Execute assigned condition for exactly 60 seconds.
- [ ] Follow script without improvisation.
- [ ] Avoid electrodes, cables, soil, pot, and damaged leaves.
- [ ] Stop event marker.

## During Recovery

- [ ] Participant exits silently.
- [ ] No room entry during recovery.
- [ ] Confirm sensors continue logging.

## After Session

- [ ] Stop and save all streams.
- [ ] Confirm files exist and sizes are plausible.
- [ ] Record electrode impedance.
- [ ] Complete metadata JSON.
- [ ] Mark quality flag: `PASS`, `WARN`, or `FAIL`.
- [ ] Add factual notes only.
- [ ] Back up raw files and checksums.
