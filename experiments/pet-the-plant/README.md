# Pet The Plant

Pet the Plant is the first Planet_Earthing experiment. It tests whether a plant's physiological signals differ during standardized interaction with a familiar host versus unfamiliar humans.

## One-Sentence Version

A plant is not assumed to be sentient; it is treated as a living sensor-rich organism whose electrical, mechanical, chemical, and water-status signals can be measured under carefully controlled human interaction conditions.

## Core Question

Can familiar-host sessions be distinguished from novel-stranger sessions using plant-derived sensor data after controlling for touch pressure, voice, heat, CO2, odor, timing, and sensor drift?
